# Classic Todo App

## Features:
    1. Users can login and signup with their email ids.
    2. Logged in users can add items to their todo lists.
    3. Users can delete any item.
    4. Users can log out whenever they want.

## Tech Stack:

    1. ReactJS
    2. CSS
    3. Bootstrap
    4. Node.JS
    5. Express.JS
    6. MongoDB
